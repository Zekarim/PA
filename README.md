# Visualization and Analysis of Personal Data Traces

This repository contains code and resources for the Projet d'approfondissement of the Master of Science HES-SO in Engineering, with an orientation in Data Science. The project focuses on two different parts, the first being the visualization and analysis of personal data traces, and the second being the prediction of video ratings using the Tournesol dataset.

## Repository Structure

The repository is organized into two main folders, Tournesol and Hestia.ai, each focusing on a specific field. The Tournesol folder contains code and notebooks related to video rating prediction using the Tournesol dataset, while the Hestia.ai folder contains code to fork a directus instance in their case.

Please refer to the respective README files inside each folder for more information on the project, dataset, algorithms, and implementation details.
